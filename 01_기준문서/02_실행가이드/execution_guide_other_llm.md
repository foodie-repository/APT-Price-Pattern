# 다른 LLM 실행 가이드

이 문서는 `Claude`, `Gemini` 등 다른 LLM에게 동일한 분석을 단계별로 요청할 때 사용하는 복사/붙여넣기용 가이드다.

이 문서는 [execution_guide.md](/Users/foodie/myproject/Real-Estate-Investment/APT-Price-Pattern/01_기준문서/02_실행가이드/execution_guide.md)의 파생 문서다.
기준 가이드가 바뀌면 이 문서도 함께 갱신한다.

## 사용 원칙

- 공통 기준 문서는 [prompt_final.md](/Users/foodie/myproject/Real-Estate-Investment/APT-Price-Pattern/01_기준문서/01_프롬프트/prompt_final.md)와 [checklist_execution.md](/Users/foodie/myproject/Real-Estate-Investment/APT-Price-Pattern/01_기준문서/02_실행가이드/checklist_execution.md)다.
- 실행 기준은 [prompt_final.md](/Users/foodie/myproject/Real-Estate-Investment/APT-Price-Pattern/01_기준문서/01_프롬프트/prompt_final.md) 단일 문서로 고정한다.
- [prompt_codex.md](/Users/foodie/myproject/Real-Estate-Investment/APT-Price-Pattern/90_참고보관/03_기준문서_구버전/prompt_codex.md), [prompt_claude.md](/Users/foodie/myproject/Real-Estate-Investment/APT-Price-Pattern/90_참고보관/03_기준문서_구버전/prompt_claude.md), [prompt_gemini.md](/Users/foodie/myproject/Real-Estate-Investment/APT-Price-Pattern/90_참고보관/03_기준문서_구버전/prompt_gemini.md)는 참고 이력으로만 두고 실행 기준으로 사용하지 않는다.
- 매매 대표가격만으로 계산한 값은 `수익률`이 아니라 `가격상승률`, `가격하락률`, `가격변화율`로 표기하고, 투자금 개념이 들어갈 때만 `갭투자 수익률`처럼 별도 용어를 쓴다.
- 다른 LLM도 `1번부터` 시작한다.
- 다만 `1번`, `2번` 단계는 재설계가 아니라 `현재 구조 재확인`이 목적이다.
- 이미 만들어진 공통 참조 파일은 특별한 오류가 없으면 덮어쓰지 않는다.
- 급지표 CSV는 공통 참조 파일이므로 [02_데이터/02_참조](/Users/foodie/myproject/Real-Estate-Investment/APT-Price-Pattern/02_%EB%8D%B0%EC%9D%B4%ED%84%B0/02_%EC%B0%B8%EC%A1%B0)에 있는 최신본을 사용한다.
