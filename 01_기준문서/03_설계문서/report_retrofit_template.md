# 기존 보고서 보강 템플릿

이 문서는 이미 작성된 `3~6단계 보고서`를 새 기준에 맞춰 보강할 때 쓰는 템플릿이다.

목적:
- 기존 보고서를 전면 폐기하지 않고, `투자 멘토 시스템` 기준에 맞게 구조를 보강한다.
- 모든 보고서가 `장기 역사 -> 현재 국면 -> 향후 시나리오 -> 투자자 대응` 흐름을 공유하게 만든다.
- `현재 국면 진단`을 약화하지 않고, 오히려 장기 패턴과 연결해 더 강하게 만든다.

---

## 1. 공통 원칙

- 기존 수치와 사실을 유지하되, 구조와 맥락을 보강한다.
- 이미 계산된 내용이 있으면 재계산보다 `해석 정렬`을 우선한다.
- 대표 사례는 설명용 예시일 뿐이고, 전국 전체 커버를 대체하지 못한다.
- 전국 단위 보고서는 본문 또는 부록에서 `전국 전체 시도/시군구`를 최소 한 번 이상 표, 랭킹, 분류표 형태로 모두 다뤄야 한다.
- 아래 4가지는 모든 핵심 보고서에 공통으로 들어간다.
  - 장기 역사 요약
  - 현재 국면 진단
  - 과거 유사 국면 비교
  - 시나리오별 투자자 대응

---

## 2. 공통 기본 구조

모든 핵심 보고서는 가능하면 아래 구조를 따른다.

### 2.1 문서 목적

- 이 문서가 무엇을 설명하는지 한 줄로 정의한다.
- 예:
  - 매매: `장기 가격 확산 패턴과 현재 국면의 강약`
  - 임차: `실수요 지지, 월세화, 임차 구조 변화`
  - 정책: `정책국면이 가격·거래·미분양 물량·공급에 미친 영향`
  - 신규 준공 간접지표: `공급 부담이 향후 6~12개월에 주는 압력`

### 2.2 분석 기준

- 기간
- 기준월
- 핵심 뷰와 테이블
- 적용 지역 범위
- 데이터 한계

### 2.3 전국 커버 요약

- 전국 전체 시도/시군구를 최소 한 번 이상 커버하는 표 또는 부록을 둔다.
- 본문은 `전국 요약 -> 권역별 정리 -> 대표 사례 심화` 순서를 기본으로 한다.
- 서울·수도권 사례가 많더라도, 지방광역시와 주요중소도시의 판정 결과는 빠짐없이 남긴다.
- 서울·수도권 장문판은 `강남/서초/송파/용산`, `마포/성동`, `노원/동대문/은평` 등 핵심 축을 본문에서 직접 설명한다.
- 지방광역시와 기타지방도 `왜 그런 판정이 나왔는지`를 주요 도시 단위로 적고, 숫자만 적고 끝내지 않는다.

### 2.3 장기 역사 요약

- `2006~현재`를 몇 개 국면으로 나눠 핵심 전환점을 정리한다.
- 이 문서 축에서 반복된 패턴과 예외 패턴을 같이 적는다.

### 2.4 데이터로 재확인된 사실

- 시장 통념이 아니라, 이번 데이터에서 분명히 확인된 사실을 적는다.

### 2.5 데이터가 수정한 통념

- 흔히 믿지만 이번 데이터가 수정하거나 약화한 통념을 적는다.

### 2.6 놓치기 쉬운 패턴

- 평균값 아래 숨어 있는 예외
- 생활권 내부 차이
- 국면 전환기의 신호

### 2.7 현재 국면 진단

- 최근 `24~36개월` 기준으로 지금 어디에 와 있는지 설명한다.
- `현재 국면 진단`은 별도 핵심 섹션으로 둔다.

### 2.8 과거 유사 국면 비교

- 최소 `2개` 이상 비교한다.
- 공통점 / 차이점 / 이번 국면만의 예외를 분리한다.

### 2.9 향후 시나리오

- `상승 / 보합 / 하락` 3가지 기본 시나리오로 정리한다.
- 각 시나리오에 대해:
  - 촉발 조건
  - 무효화 조건
  - 먼저 확인할 지표

### 2.10 투자자 대응

- `우선 매수 검토`
- `관찰 유지`
- `보수 접근`
- `회피`

이 4구간으로 나눠 정리한다.

### 2.11 한계와 추가 확인 과제

- 표본 부족
- 조인 한계
- 정책 중첩
- 지역별 예외
- 추가 데이터 필요성

### 2.12 전국 부록

- 가능하면 별도 부록 또는 CSV 링크로 `전체 시군구 판정표`, `권역별 랭킹표`, `후보/관찰/회피 전수표`를 붙인다.
- 본문에 들어가지 못한 지역도 부록에서는 빠지지 않게 한다.

## 3. 문서별 보강 포인트

## 3.1 매매 시장 분석

반드시 추가할 것:
- 장기 국면별 가격 확산 순서
- 현재 국면과 유사한 과거 국면 최소 2개
- `이미 알고 있던 사실 / 데이터가 수정한 통념 / 놓치기 쉬운 패턴`
- `우선 매수 검토 / 관찰 유지 / 보수 접근 / 회피` 후보
- 수도권과 지방의 해석 체계 차이
- 전국 전체 시군구를 커버하는 `대응 구분 전수표` 또는 권역별 랭킹표

있으면 좋은 것:
- 같은 생활권 안 상품성 차이
- 호가-실거래 긴장도
- `준공후 미분양` 리스크

## 3.2 임차 시장 분석

반드시 추가할 것:
- 장기 임차 사이클
- 전세와 월세의 다른 속도
- 현재 임차 국면 진단
- `매매보다 임차가 강한 지역`, `월세화 리스크 지역`, `기다려야 할 지역`
- 전국 전체 시군구를 커버하는 `임차 유형 전수표`

있으면 좋은 것:
- 비아파트 공급 버퍼
- 임대 운용 방식별 차이

## 3.3 정책 영향 분석

반드시 추가할 것:
- 과거 정책국면 비교
- 현재 정책국면의 제약과 완충
- `정책명`보다 `정책이 나온 배경`과 `직전 미분양·공급 구조`
- 투자자 입장에서 어떤 행동 변화를 요구하는지
- 전국 전체 시군구를 커버하는 `정책 후 판정표`

있으면 좋은 것:
- `낮은 준공후 미분양 압력 + 강한 임차 수요 기반` 후보
- 규제 민감 지역 vs 완화 수혜 후보

## 3.4 신규 준공 간접지표 분석

반드시 추가할 것:
- 장기 공급 사이클
- 현재 공급부담의 위치
- `0~6개월`과 `6~12개월` 시차 구분
- 신규 준공 간접지표가 맞는 곳과 틀리는 곳의 예외 패턴

있으면 좋은 것:
- `공급이 많아도 오르는 곳`, `공급이 적어도 약한 곳` 규칙화

---

## 4. 최종 종합보고서로 넘길 때의 공통 출력

기존 3~6단계 보고서를 보강한 뒤에는 아래 내용을 종합보고서로 넘긴다.

- 각 보고서의 핵심 사실 3~5개
- 현재 국면 진단 핵심 문장
- 과거 유사 국면 2개 비교 요약
- 상승/보합/하락 시나리오별 핵심 조건
- `우선 매수 검토 / 관찰 유지 / 보수 접근 / 회피` 후보
- 투자자가 추가로 확인해야 할 데이터

---

## 5. 최종 투자 멘토 노트로 넘길 항목

기존 보고서 보강 후, 아래는 별도 멘토 노트 또는 플레이북 문서로 옮긴다.

- 지금 무엇을 사야 하는가
- 지금 무엇을 피해야 하는가
- 어떤 질문을 더 해야 하는가
- 어떤 데이터가 더 필요할 때 판단이 달라지는가
- 단지 단계로 내려갈 때 무엇을 먼저 확인해야 하는가

---

## 6. 사용 시점

- 이 템플릿은 `7. 예측 및 검증` 전에는 구조 확인용으로만 쓴다.
- 실제 본격 보강은 `7번 결과`가 나온 뒤 `8. 보고서 작성` 단계에서 적용한다.
- 예측 점수와 과거 데이터 검증 기준은 `01_기준문서/03_설계문서/prediction_validation_design.md`를 따른다.
