# 토지거래허가구역 최신 웹 확인 메모

목적:
- DuckDB `토지거래허가구역`은 허가 승인/수집 이력 보조 데이터로 보고, `현재 활성 토허구역` 자체는 최신 공식 웹 공고로 재확인한다.
- 이 문서는 활성 구역의 확정 마스터가 아니라, 분석 직전 확인해야 할 공식 소스의 출발점 메모다.
- 구조화된 현재 참조표는 `02_데이터/02_참조/토지거래허가구역_활성참조_20260313.csv`를 함께 본다.

원칙:
- 분석 기준 시점마다 최소 1회 이상 최신 웹 검색을 수행한다.
- 공식 최신 정보는 서울시, 경기도/시군, 국토교통부, 정책브리핑 같은 1차 공공 소스를 우선 사용한다.
- DuckDB와 최신 웹 정보가 다르면 분석 기준 시점에 더 가까운 공식 최신 정보를 우선한다.
- 토허 지역은 허가 승인 절차 때문에 최근 `1.5~2개월` 실거래 반영이 늦을 수 있으므로, 현재시장 해석 시 `실거래`, `허가 승인 데이터`, `호가/매물`, `정책 공고 시점`을 함께 본다.

실무 메모:
- 서울은 `강남구`, `서초구`, `송파구`, `용산구`처럼 구 단위로 보지 말고 실제 공고에 적힌 `동/아파트지구/재건축 구역` 범위를 확인한다.
- 경기도는 시군 전체가 아니라 특정 동·지구·사업지 위주로 지정되는 경우가 많으므로, 시군구 단위 집계만으로 활성 여부를 단정하지 않는다.
- 향후 분석에서 최신 웹 확인 결과를 구조화하려면 별도 수동 입력 CSV 또는 DuckDB 보조 테이블을 두고 `기준일`, `시도`, `시군구`, `세부구역명`, `활성여부`, `출처URL`을 기록한다.
