## 2. 이미 DuckDB에 있지만 아직 활용 고도화가 더 필요한 축

| --- | --- | --- | --- |
| 분양 실적 보강 | `KOSIS_분양실적` | 적재 완료 | 실제 분양 호수 흐름을 HUG 초기분양률과 같이 볼지 정리 필요 |
| 분양 보증 보강 | `KOSIS_분양보증현황`, `KOSIS_분양보증세대수_발급`, `KOSIS_분양보증세대수_잔액` | 적재 완료 | 공급 기대와 보증 흐름을 1차 시장 보조축으로 쓸지 검토 필요 |
| 분양권 / 전매 | `분양권전매` | 적재 완료 | 과열, 회전율, 투자 수요 조기 감지 축으로 쓸지 검토 필요 |
| 구매부담 보강 | `KB_PIR`, `KB_주택구매력지수`, `KB_주택구입잠재력지수` | 적재 완료 | 전국 점수화보다 수도권 보조 해석으로 쓸지 정리 필요 |
| 가격 보조 | `KB_매매가격지수`, `KB_매매가격지수_증감률`, `KB_전세가격지수`, `KB_전세가격지수_증감률`, `KB_전세가율`, `KB_월세가격지수`, `KB_전세수급지수`, `KB_전세거래활발지수`, `KB_전세가격전망지수` | 적재 완료 | KOSIS/실거래와의 역할 분리, 보조축 위치 정리 필요 |
| 토지 / 보조시장 | `KOSIS_지가변동률` | 적재 완료 | 아파트 직접 지표가 아닌 보조 국면 변수로 쓸지 검토 필요 |

- `이미 DuckDB에 있다`는 사실과 `현재 분석에서 잘 쓰고 있다`는 사실은 다르다.
