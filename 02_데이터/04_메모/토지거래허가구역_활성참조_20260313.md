# 토지거래허가구역 활성 참조 요약

참조 파일:
- `02_데이터/02_참조/토지거래허가구역_활성참조_20260313.csv`

목적:
- 최신 공식 웹 확인 결과를 `현재 활성 여부` 관점에서 구조화한 수동 참조표다.
- DuckDB `토지거래허가구역`의 허가 승인/수집 이력과 별도로, `현재 활성 토허구역`을 읽기 위한 보조 기준으로 쓴다.

상태 해석:
- `활성확인`: 공식 현황 페이지나 공식 공고에서 현재 지정기간이 분석 기준일을 포함함을 확인한 경우
- `활성추정_공고확인`: 공식 재지정 공고는 확인했지만, 현재 웹에서 첨부 공고문 세부 범위나 종료일을 바로 확인하지 못한 경우
- `종료확인`: 공식 해제 공고 또는 종료 사실을 확인한 경우
- `추가확인필요`: 최신 공식 공고를 아직 더 확인해야 해서 현재 활성 여부를 단정하면 안 되는 경우

실무 사용 원칙:
- `활성확인`만 현재 토허구역 해석의 1차 기준으로 사용한다.
- `활성추정_공고확인`은 보수적으로 `토허 영향 가능 지역`으로만 메모하고, 강한 결론에 쓰지 않는다.
- `추가확인필요`는 분석·예측 점수에 바로 반영하지 말고, 공고문 원문이나 지자체 최신 공지를 추가 확인한 뒤 확정한다.
- 서울은 `구 단위`보다 `아파트 부지`, `재건축 대상`, `신속통합기획`, `모아타운`처럼 실제 공고 범위가 더 중요하다.
- 경기도는 시군 전체가 아니라 특정 동·사업지구 지정이 많아서, `시군구 단위 활성`로 단정하지 않는다.

현재 요약 메모:
- 서울시는 `2026-02-17 현재` 공식 지정현황 페이지가 있고, 아파트 시장 해석에 직접 영향이 큰 항목은 `강남·서초·송파·용산구(아파트 부지)`, `서울시 전체 아파트`, `재건축 14개 아파트`, `외국인 등 주택 거래 토허`다.
- 경기도는 `2025-07-05 기준` 지정현황 페이지와 일부 개별 재지정/해제 공고를 확인했다.
- 경기도의 경우 `광명 하안구역`, `하남 기업이전부지`, `1기 신도시 상가`는 종료확인으로 정리했고, `고양 대곡역세권`, `시흥 시가화 예정지구`는 재지정 공고는 확인했지만 세부 기간은 공고문 원문 재확인이 필요하다.
