# BOK 금리·대출 지표 정의와 활용 메모

## 1. 이번에 DuckDB에 반영된 테이블

- `KOSIS_ECOS_기준금리`
- `KOSIS_BOK_대출금리_신규`
- `KOSIS_BOK_대출금리_잔액`
- `KOSIS_BOK_가계대출_업권별`
- `KOSIS_BOK_가계대출_용도별`
- `KOSIS_BOK_가계대출_지역별`
- `KOSIS_BOK_대출수요`
- `KOSIS_BOK_대출태도`
- `KB_아파트담보대출PIR`

## 2. 핵심 해석 원칙

- 금리와 대출은 `시군구 직접 랭킹 지표`보다 `전국 공통 금융조건`과 `시도 보조축`으로 쓰는 편이 맞다.
- 금리는 `기준금리`보다 실제 차주가 체감하는 `신규취급 주담대금리`, `전세자금대출금리`, `가계대출금리`가 우선이다.
- `기준금리`는 실제 체감금리를 대체하지 않지만, `인상기/동결기/인하기`와 같은 정책 전환의 배경축으로는 반드시 같이 본다.
- 대출은 절대 잔액보다 `최근 6개월 변화`, `전년동기 변화`, `주담대와 기타대출의 엇갈림`이 더 중요하다.
- `대출수요`와 `대출태도`는 실거래보다 선행할 수 있지만, 지역별 종목 선택보다 `금융여건의 온도`를 설명하는 보조축으로만 쓴다.
- `KB_아파트담보대출PIR`은 수도권 3개 시도(`서울`, `경기`, `인천`)만 제공되므로 전국 공통 지표로 확장하지 않는다.

## 3. 이번 단계에서 바로 쓰는 지표

### 3.1 금리

- 기준금리
- 신규 가계대출금리
- 신규 주택담보대출금리
- 신규 전세자금대출금리
- 신규 고정형/변동형 주담대금리
- 잔액 주택담보대출금리
- 잔액 가계대출금리

### 3.2 대출 잔액

- 예금취급기관 가계대출 총잔액
- 예금은행 가계대출 총잔액
- 예금취급기관 주택담보대출 잔액
- 예금은행 주택담보대출 잔액
- 예금취급기관 기타대출 잔액
- 정책대출 잔액

### 3.3 시도 보조축

- 예금은행 가계대출 잔액의 시도별 전년동기 증가율
- 예금은행 주담대 잔액의 시도별 전년동기 증가율
- 예금은행 가계대출 대비 주담대 비중

### 3.4 금융여건 보조축

- 국내은행 차주가중 종합 `대출수요`
- 국내은행 차주가중 종합 `대출태도`
- `KB_아파트담보대출PIR`(서울·경기·인천)

## 4. 현재 데이터로 읽은 핵심 시그널

- `2026-02` 기준금리는 `2.50%`다.
- `2026-01` 신규 주택담보대출금리는 `4.29%`, 전세자금대출금리는 `4.06%`다.
- 신규 주담대금리는 최근 6개월 `+0.33%p` 올라, 1년 전보다 크게 낮아진 상태라고 보긴 어렵다.
- 기준금리 `2.50%`와 신규 주담대금리 `4.29%`를 같이 보면, 정책금리는 이미 인하 국면에 들어섰더라도 실제 차주가 체감하는 자금조달 여건은 아직 완전히 완화됐다고 보기 어렵다.
- `2025-12` 예금취급기관 주택담보대출 잔액은 전년동기 대비 `+7.5%`, 기타대출은 `-2.2%`다.
- 즉 현재 레버리지 확대는 `광범위한 신용 팽창`보다 `주담대 중심의 선별 회복` 성격이 강하다.
- `2025.04Q` 대출수요 종합지수는 `6.0`, 대출태도 종합지수는 `-21.0`이다.
- 차주 수요는 약한 플러스지만 은행 태도는 여전히 보수적이므로, `돈이 너무 쉬워서 올라가는 장`으로 읽으면 과하다.

## 5. 분석 보고서에 붙이는 위치

### 5.1 장기 패턴 라이브러리

- `초저금리기`, `금리충격기`, `동결기`, `인하기`, `고금리·선별 레버리지 확장기`를 구분하는 배경 변수로 사용
- 현재 국면이 왜 `2020~2021`과 다른지 설명하는 축으로 사용

### 5.2 매매시장분석

- `거래 회복의 질` 다음 레이어로 사용
- 왜 서울이 기대 대비 거래가 약한지, 왜 경기·인천이 상대적으로 동행 회복하는지 설명
- `주담대 중심 회복`과 `기타대출 둔화`를 함께 적어 과잉 해석을 막음

### 5.3 임차시장분석

- 월세 비중 상승의 금융 배경 설명
- 전세 둔화가 단순 수요 약화인지, 금리 부담 전이인지 구분하는 보조축

### 5.4 정책영향분석

- `기준금리` 전환과 `대출규제`를 정책 이벤트로만 다루지 않고 실제 금융비용과 공급태도까지 같이 설명
- 정책 완화가 나와도 금리와 은행 태도가 상쇄하는지 확인

### 5.5 최종 통합 보고서

- 현재 시장을 `저금리 동조화장`이 아니라 `고금리·선별 레버리지 확장기`로 정의하는 근거

## 6. 당장 쓰지 않는 항목

- `고정/변동금리 비중`: 현재는 금리 수준만 있고 비중은 없다.
- `대출수요/대출태도`의 세부 항목 분해: 종합지수는 바로 쓰되, 세부 차주군 분해는 다음 단계로 넘긴다.

## 7. 체크리스트

- [x] 신규/잔액 대출금리의 최신 기준월과 최근 6개월·12개월 변화를 확인했다.
- [x] 가계대출 업권별·용도별·지역별 잔액의 최신 기준월과 증가율을 확인했다.
- [x] 대출수요와 대출태도를 금융여건 보조축으로 사용할 수 있는지 확인했다.
- [x] `KB_아파트담보대출PIR`은 수도권 보조축으로 제한 사용하기로 정했다.
- [ ] 임차시장 보고서에 금리·전세자금대출 연결 해석을 추가한다.
- [ ] 예측/검증 단계에 금리·대출을 직접 점수화할지, 해석 보조축으로만 둘지 확정한다.
