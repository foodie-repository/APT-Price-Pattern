# KOSIS 추가지표 도입 검토 체크리스트

작성일: 2026-03-14

## 목적
- 이 문서는 현재 프로젝트에 없는 경제/부동산 보조지표를 `한 번에 전부` 붙이지 않고, `우선순위별로 하나씩` 검토하기 위한 운영 문서다.
- 기준 자료는 외부 폴더의 `KOSIS_통계목록.csv`, `부동산시장분석_필요통계지표.md`이며, 본 문서는 그 1차 검토 의견을 현재 프로젝트 구조에 맞게 정리한 것이다.
- 목표는 `지표 수집` 자체가 아니라, 현재 분석 체계의 설명력과 판단력을 실제로 높이는 지표만 선별하는 데 있다.

## 현재 운영 기준
- 실제 원천 수집은 `Real-Estate-Data` 프로젝트에서 수행하고, 현재 프로젝트에서는 `반입 -> DuckDB 적재 -> 파생지표 생성 -> 보고서/점수/게이트 반영`을 맡는다.
- 추가 지표의 반영 기준은 `01_기준문서/03_설계문서/추가_지표_도입_로드맵_20260315.md`, `01_기준문서/03_설계문서/추가_지표_분석반영_매핑표_20260315.md`를 함께 따른다.
- 본 문서는 `후보 검토와 누락 방지 체크리스트` 역할을 유지하고, 실제 반영 위치는 위 두 설계 문서에서 고정한다.

## 기본 원칙
- 현재 이미 강한 축과 중복되는 지표는 우선순위를 낮춘다.
- 월별·분기별, 지역 단위가 있는 지표를 단기 국면 판단에 우선 사용한다.
- 연간·전국 단위 지표는 `타이밍 지표`가 아니라 `구조적 보정 변수`로 사용한다.
- `사실 / 해석 / 가설` 구분 원칙을 유지한다.
- 지표 하나를 붙일 때마다 `정의 -> 공간 단위 -> 주기 -> 현재 분석과의 연결 위치 -> 한계`를 먼저 확정한다.

## 현재 프로젝트의 강한 축
- 가격 변화율, 거래 회복, 상승 지역 비중
- 전세가율, 월세 비중, 임차 흐름
- 정책 이벤트, 토지거래허가구역, 정책 전후 비교
- 입주예정 물량, 준공후 미분양, 공급 해석
- 상품성/입지, 투자 검토 게이트, 예측/검증

## 현재 프로젝트의 부족한 축
- 실수요 유입 흐름
- 1차 분양시장 심리와 선행성
- 가계의 구매력과 레버리지 비용
- 소유구조와 지역별 공실·공동화
- 공식 심리 지표와 실제 거래의 괴리 해석 고도화

- 상세 정의와 활용 원칙은 `02_데이터/04_메모/KOSIS_아파트매매거래현황_4종_지표정의_활용메모_20260314.md`를 따른다.

- `HUG` 분양 관련 지표와 `KOSIS` 부동산 관련 소비심리지수, `KB` 민간 심리 지표는 이미 DuckDB에 반영되었다.
- `민간아파트 분양가격`, `분양가격지수`, `초기분양률`, `신규 분양세대수`, `대지비비율`은 신규 수집 대상이 아니라 `보고서/점수 반영 고도화` 대상으로 전환한다.
- `공식 심리 지표`도 신규 수집 대상이 아니라 `심리-거래 괴리`, `심리-전세 괴리`, `뉴스와 역할 분리` 고도화 대상으로 전환한다.
- `주담대금리`, `가계대출`, `대출수요·대출태도`, `KB PIR`은 신규 수집 대상이 아니라 `전국 공통 금융조건` 반영 고도화 대상으로 전환한다.
- `기준금리`와 필요 시 `고정/변동금리 비중`만 별도 DuckDB 테이블 확인 전까지 보강 후보로 유지한다.
- `매입자거주지별`, `매입자연령대별`, `거래규모별`, `거래주체별`은 이미 반영되었으므로 재수집이 아니라 `보고서/점수/게이트 활용 고도화`로 전환한다.
- 정부와 기업의 투자 방향은 단일 월별 숫자보다 `정책/기업 투자 이벤트 DB`로 병행 설계한다.
- 뉴스는 `감성점수`보다 `기사량 -> 주제 태그 -> 뉴스-거래 괴리` 순으로 활용한다.

## 4. 심리 지표

### 지표 정의
- 소비자 기대, 주택가격 전망, 기대인플레이션, 뉴스/검색 트렌드를 통해 `시장 참여자의 기대 변화`를 보는 지표다.

### 후보 통계표
- `DT_511Y002` 소비자동향조사(전국, 월)
- `DT_511Y004` 소비자동향조사(지역, 월)
- `DT_511Y003` 기대인플레이션율(전국, 월)
- 비KOSIS 보조: 뉴스, 검색트렌드, 네이버 매물 호가 심리

### 추가 필요 사유
- 현재 프로젝트는 실제 거래와 임차 지표는 강하지만, 공식 심리 시계열은 약하다.
- 심리는 회복 초입과 과열 말기를 탐지하는 데 유용하다.
- 특히 `심리만 강하고 거래가 안 따라오는 구간`을 잡아내면 허위 반등을 걸러낼 수 있다.

### 활용 위치
- 장기 패턴 라이브러리: 전환점 보조축
- 매매시장 분석: 심리-거래 괴리, 심리-전세 괴리 확인
- 정책영향분석: 정책 발표 직후 심리 반응 vs 실제 거래 반응 비교
- 과열 경고: 외곽 확산과 결합된 심리 급등 탐지

### 현재 반영 상태
- DuckDB 반영 완료:
  - `KOSIS_소비심리_부동산시장`
  - `KOSIS_소비심리_주택시장`
  - `KOSIS_소비심리_주택매매`
  - `KOSIS_소비심리_주택전세`
  - `KOSIS_소비심리_토지시장`
  - `KB_매수우위지수`
  - `KB_매매가격전망지수`
  - `KB_매매거래활발지수`
- 다음 단계는 `심리-거래 괴리`, `심리-전세 괴리`, `공식 심리와 민간 체감심리의 역할 분리`, `뉴스와 심리의 역할 분리`를 파생지표로 정리하는 일이다.

### 추가 공식 심리 후보 메모
- 현재 프로젝트에서 `공식 심리`의 본축은 이미 수집한 `KOSIS` 부동산 관련 소비심리지수다.
- 아래 후보는 그 다음에 붙일 `추가 공식 심리`로 본다.
  - `주택가격전망CSI`
  - `소비자심리지수(CCSI)`
  - `가계부채CSI`
  - `건설업 BSI`
  - `금리수준전망CSI`
  - `경제심리지수(ESI)`
- 내부 우선순위는 `주택가격전망CSI -> CCSI -> 가계부채CSI -> 건설업 BSI -> 금리수준전망CSI -> ESI` 순이 적절하다.
- `건설업 BSI`와 `ESI`는 의미는 있지만, 현재 프로젝트는 이미 `HUG 분양가격 / 초기분양률 / 분양세대수`가 있어 우선순위를 낮춰도 된다.
- `CCSI`와 `가계부채CSI`는 지역 선별 지표보다 `전국 공통 국면 변수`로 쓸 가능성이 높다.

### 주의점
- 심리는 단독 신호로 쓰면 오판 가능성이 높다.
- 반드시 거래, 전세가율, 초기분양률과 함께 봐야 한다.

### 체크리스트
- [ ] 소비자동향조사 세부 항목에 주택가격전망 관련 항목이 있는지 확인한다.
- [ ] `주택가격전망CSI`를 현재 `KOSIS_소비심리_주택매매`와 교차검증할 가치가 있는지 검토한다.
- [ ] `CCSI`를 지역 선별 지표가 아니라 전국 공통 국면 변수로 쓸지 검토한다.
- [ ] `가계부채CSI`를 실제 가계대출과 함께 볼 보조축으로 쓸지 검토한다.
- [ ] `건설업 BSI`가 이미 있는 `HUG` 분양지표 대비 추가 설명력이 있는지 검토한다.
- [ ] `ESI`가 `CCSI`와 `BSI`를 따로 보는 것보다 더 유의미한지 검토한다.
- [ ] 뉴스/검색트렌드와 공식 심리 지표의 역할 분담을 정리한다.
- [ ] 심리-거래 괴리 지표를 만들 수 있는지 검토한다.
- [ ] 과열 경고 보조축으로 사용할 임계 조건을 설계한다.
