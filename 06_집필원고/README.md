# 06_집필원고

이 폴더는 `책 출간용 원고 작업 공간`이다.

- `05_글쓰기`: 블로그, 프리미엄 콘텐츠, 브런치 등 채널별 발행 원고
- `06_집필원고`: 장기 집필, 출간 기획, 목차 설계, 독자 실습, 부록

운영 원칙:

- AI별 작업물은 하위 폴더로 분리한다.
- 공통 기준은 현재 프로젝트의 기준 문서를 따른다.
- 책 원고는 날짜 중심이 아니라 `부/장/절` 중심으로 관리한다.
- 분석 결과를 그대로 복사해 누적하지 말고, 필요한 결과만 요약하고 원천 산출물 경로를 남긴다.
- 목표는 `분석 결과 소개`가 아니라 `독자가 투자 멘토 시스템을 따라 만들 수 있게 설명하는 것`이다.

현재 생성된 하위 폴더:

- `06_집필원고_codex`

향후 필요하면 아래처럼 병렬 확장할 수 있다.

- `06_집필원고_claude`
- `06_집필원고_gemini`
