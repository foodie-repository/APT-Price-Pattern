# 06_집필원고_codex

이 폴더는 Codex가 작성하는 `종이책 원고 전용 작업 공간`이다.

## 목표

- 독자가 이 저장소의 사고방식과 분석 흐름을 따라가며
- `DuckDB 기반 부동산 분석 시스템`을 직접 만들고
- 최종적으로 `투자 멘토 시스템`의 축소판 또는 실전형 기초판을 재현할 수 있게 돕는다.

## 이 폴더에서 다룰 것

- 출간 기획
- 독자와 문제 정의
- 전체 목차와 장별 메시지
- 실제 본문 원고
- 노트북 기반 시각화 및 검증 원고
- 독자 실습 설계
- 도표 및 표 기획
- 용어/데이터/뷰 사전

## 이 폴더에서 다루지 않을 것

- 채널별 발행 원고
- 날짜 중심의 분석 결과 적재
- 원천 데이터 자체 저장

## 집필 원칙

- 책의 중심 질문은 `어떻게 따라 만들 것인가`다.
- 결과보다 `재현 가능한 절차`를 우선한다.
- 각 장은 `왜 필요한가 -> 무엇을 보는가 -> 어떻게 따라하는가 -> 어떻게 해석하는가` 구조를 유지한다.
- 결론은 가능하면 `사실 / 해석 / 가설`을 유지한다.
- 독자가 혼동하기 쉬운 지점은 `흔한 오해` 섹션으로 별도 정리한다.
