# 시각자료 목록

책에서 우선 검토할 시각자료 초안 목록이다.

## 1. 시스템 개요

- 투자 멘토 시스템 전체 흐름도
- 데이터 입력 -> 전처리 -> 분석 -> 해석 -> 행동 라벨 다이어그램

## 2. 시장 읽기

- 2006년 이후 장기 국면 구분 타임라인
- 수도권 vs 지방광역시 vs 기타지방 흐름 비교
- 상급지 선행 -> 중급지 확산 -> 하급지 지연 예시 도식

## 3. 임차/정책/공급

- 전세가율과 월세 비중 비교 차트
- 정책 이벤트와 시계열 겹침 예시
- 입주예정 물량과 준공후 미분양 해석 차트

## 4. 후보 스크리닝

- 후보 축소 과정 도식
- `우선 검토 / 판단 보류 / 즉시 제외` 판정 트리
- 저평가 신호 vs 과열 신호 2축 매트릭스

## 5. 독자 실습

- 독자가 따라해야 할 월별 루틴 1페이지 요약
- 초보자용 데이터 점검 체크리스트 시각화

## 6. 노트북 기반 검증 차트

- 장기 대표가격 흐름 재확인용 라인 차트
- 수도권/지방광역시/기타지방 비교 차트
- 급지 간 격차 확인 차트
- 후보 지역의 거래회복, 전세가율, 공급 부담 검증 차트
- 단지 예시 drill-down 차트

## 7. 메모

- 각 시각자료는 `설명용`, `실습용`, `결과요약용`, `검증용` 중 목적을 먼저 정한다.
- 복잡한 차트보다 `판단 구조가 보이는 도식`을 우선한다.
- 노트북 기반 차트는 향후 `04_결과/04_노트북/` 산출물과 연결한다.
