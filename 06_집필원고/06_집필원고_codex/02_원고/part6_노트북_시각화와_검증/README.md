# part6_노트북_시각화와_검증

이 파트는 `보고서 중심 분석` 다음 단계로, 수치를 차트로 다시 검증하는 노트북 기반 작업을 책 원고에 포함하기 위한 자리다.

## 이 파트의 역할

- 보고서의 핵심 결론을 차트로 다시 점검하는 방법 설명
- 표로는 잘 안 보이는 패턴을 시각화로 확인하는 방법 정리
- 노트북과 보고서의 역할 차이 설명
- 향후 실제 노트북 파일과 연결될 원고 구조 확보

## 현재 상태

- 아직 시각화 코드와 노트북 예시는 작성하지 않았다.
- 우선은 `무슨 차트를 왜 그릴지`를 먼저 구조화한다.
