## 권장 파일명

- `ch01_왜_대부분의_투자판단은_흔들리는가.md`
- `ch02_이책이_만들려는_최종결과물.md`

처럼 `chNN_제목.md` 형식을 우선 사용한다.

## 초안 운영 방식

- 한 파일에 한 챕터를 원칙으로 한다.
- 아직 쓰지 않은 장은 빈 파일보다 `TODO` 메모를 먼저 만든다.
- 중복 서술이 많아지면 `05_부록`이나 `00_출간기획`으로 올린다.
- 노트북 시각화 파트는 실제 코드가 나오기 전까지 `무슨 차트를 왜 그리는가` 중심으로만 구조를 잡는다.
