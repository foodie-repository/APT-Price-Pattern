# 프로젝트: 아파트 가격 변화 패턴 분석

### 1. 목적: 지역간/지역내 아파트 단지별 매매 및 전월세 가격 변화 패턴 분석
- 가격 확산 경로와 변화 패턴을 분석하여 투자 인사이트를 도출한다.
- 도출된 인사이트를 토대로 현재 시장상황과 정부정책 그리고 가격을 분석하여, 앞으로 상승 또는 하락할 지역과 단지를 확률적(통계적/데이터 기반)으로 도출한다.

### 2. 데이터: DuckDB 저장
- **[수정 권장]** DB 파일 경로 및 테이블명 등 명시: (예: `data/real_estate.duckdb`, `raw_transaction` 테이블 등)
- 실거래가 데이터 (매매, 전월세)
- 공종주택_전국: 아파트만 사용

### 3. 데이터 전처리
#### 1) 전용면적 분류 (한국부동산원 기준 5단계)
- 컬럼명: `전용면적_구분`
- 초소형: 전용면적 40㎡ 이하
- 소형: 전용면적 40㎡ 초과 ~ 60㎡ 이하
- 중소형: 전용면적 60㎡ 초과 ~ 85㎡ 이하
- 중대형: 전용면적 85㎡ 초과 ~ 135㎡ 이하
- 대형: 전용면적 135㎡ 초과

#### 2) 추정평형 계산 및 평형대 분류 (7단계)
- 추정평형 = `전용면적(㎡) x 0.4`
- 컬럼명: `평형대_구분`
- 분류: 10평 미만, 10평대, 20평대, 30평대, 40평대, 50평대, 60평 이상

#### 3) 주소 파싱
- 원본 시군구 컬럼(예: "서울특별시 강남구 압구정동")을 `시도` / `시군구` / `읍면동`으로 분리
- 세종특별자치시는 시군구에 '세종시'로 대체

#### 4) 날짜/연식 컬럼 생성
- 계약연도 = 계약년월 / 100 (정수형)
- 연식 = 현재년도 - 건축년도 (최소 1년)
- 컬럼명: `연식_구분`
- 분류: 5년 미만, 5~10년 미만, 10~20년 미만, 20~30년 미만, 30년 이상 **([수정 적용] 구간 경계값 이상/미만 명확화)**

### 4. 분석 내용
#### 1) 대한민국 부동산 시장의 특징 (검증할 가설)
- 대한민국 부동산 시장은 사이클이 존재한다. (사이클의 중요성)
- 부동산 시장의 특징은 입지에 따라 상급지부터 가격이 상승하기 시작하여 이어서 중급지, 하급지 순으로 가격 상승 패턴을 보인다. (입지의 중요성)
  - **[급지 판단 기준]** 지역/단지의 상급지, 중급지, 하급지 구분은 외부 지표가 아닌 실제 데이터의 **가격 수준**을 기준으로 분류 및 판단한다.
- 가격 변화는 인접 지역 간 연계된 패턴을 보인다.
- 동일 지역 내에서도 교통 접근성, 학군, 연식, 세대수 등 상품성에 따라 가격이 상승하는 패턴이 다르다. (상품성의 중요성)
- 갭벌리기, 갭메우기, 풍선효과와 같은 특징을 보인다.

#### 2) 실거래 데이터 기반 패턴 분석
- 실거래 데이터는 매수자와 매도자의 최종 의사결정의 결과이다.
- 가격 데이터를 통한 가격 변동 패턴 분석은 매매와 전월세 모든 데이터를 분석하고, 연관성을 분석한다.
- 단, 데이터 분석 정합성 확보를 위해 매매 데이터에서 **직거래와 해제 건은 제외**한다.
- 데이터 분석 시 동일 단지여도 면적이 차이가 있으므로, 전용면적 분류기준에 따라 그룹별로 분석한다.
- '1) 대한민국 부동산 시장의 특징'에서 언급된 보편적/경험적 근거를 데이터 기반으로 확인/검증(통계 추이, 시계열 등)한다.
- '1) 대한민국 부동산 시장의 특징'에서 언급된 보편적/경험적 근거 외에 데이터 기반으로 패턴을 분석하여 상식으로 알려진 것 외에 유의미한 패턴을 발견한다.
- 부동산 시장의 특성 상 정부정책(지역규제, 세금규제, 대출규제 등)에 따라 지역적으로 가격 변동 패턴이 다르게 나타난다. 정부정책에 따른 지역별 가격 변동 패턴을 데이터 기반으로 분석한다.
- 지역별 영향이 아닌 공통적 영향인자 (환율, 금리 등)는 가격 변동 패턴 분석에서 제외한다.
- 특정 시점에 특정 지역에서 입주물량이 증가할 경우 가격 변화에 미치는 영향을 분석한다. (단기적/중장기적 영향 구분)
  - **[입주물량 산출 기준]** 외부 입주물량 데이터를 참고하지 않고, 실거래 데이터나 공동주택 데이터 내의 **건축년도**(또는 사용승인일, 완공일 등 관련 컬럼)를 기준으로 해당 시점의 자체 입주물량을 산출해 사용한다.
- 과거 정부정책은 `02_데이터/04_메모/정책_이벤트_매뉴얼.md`를 참고한다.
- 새로 업데이트 되는 정부정책은 `/Users/foodie/myproject/Real-Estate-Data/수요분석/정책_이벤트_매뉴얼.md`를 참고한다.
- `90_참고보관/01_사이클투자`에 있는 내용은 부동산 시장의 오랜 경험을 바탕으로 정리한 내용이다. 전체 흐름을 분석하는 과정에서 참고하되, 절대원칙으로 삼아서는 안된다.
- 데이터 분석을 통해 도출된 결과를 바탕으로 `90_참고보관/01_사이클투자`에 있는 내용을 비판적으로 검증한다.
