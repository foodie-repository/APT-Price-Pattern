# 아파트값 5차 파동 특강 - 4교시 정리

## 📌 핵심 주제: 6차 파동의 실전 — 침체·회복·확산·급등·위축의 전 과정

---

## 1. 침체기 (2010~2013년)

### 금융위기 이후 시장 상황
- 2008년 9월 15일 금융위기 → 시장 위축
- 2009년 9월 수도권 DTI 규제 + 보금자리주택 발표 → 시장 본격 침체
- 2010년부터 침체기로 진입

### 침체기에도 강남은 움직인다
- **2010년 2월**: "강남 아파트만 들썩" 기사 등장
- 침체장이라고 해서 강남이 폭락하지 않음
- 좋은 입지는 **움직일 기미만 있으면 가장 먼저 반응**
- 재건축 아파트도 침체장에서 상승 기사 다수

### 전세가격 상승 시작
- 2011년 8월: 수도권 전세가격 5년 만에 매매가 절반(50%) 돌파
- 2012년: 서울 주택가격 하락세 → "하우스푸어" 용어 등장
- **"집값은 이제 바닥이다" vs "상승은 어렵다"** — 시장 심리 극도로 혼란

---

## 2. 회복기 (2013~2016년)

### 회복장의 신호 (1차~4차 파동과 동일한 패턴)
1. **전세가율 상승** (60% → 70% → 일부 90%)
2. **세금 규제 완화** (양도세 면제 등)
3. **청약제도 개편**
4. **신도시 개발 중단** (공급 축소 신호)

### 주요 사건 타임라인

| 시기 | 사건 | 의미 |
|------|------|------|
| 2013.1 | 월세 전환 급증 | 전세 공급 부족 심화 |
| 2013.4 | 양도세 면제 대책 (5년간) | 정부의 시장 살리기 시작 |
| 2014.1 | 청약제도 개편 | 1주택자 우대, 일시적 2주택 완화 |
| 2014.9 | 9.1 부동산 대책 | 신도시 개발 중단, LTV·DTI 완화 |
| 2014~15 | 일부 지역 전세가율 90% 돌파 | "차라리 집 사는 게 낫다" 심리 형성 |

### 아무도 모르는 상승의 시작
- 분양시장 경쟁률 상승 → 매매시장은 아직 정체
- **"상승장의 신호는 전체 시장 상승이 아니라, 분양시장의 경쟁률 상승"**
- 신축 선호 → 미분양 감소 → 분양가 상승 → 기축 동반 상승 (수년 후)
- 2014년: 가양동 모아파트 전세 방문에 10명씩 줄 서는 상황

### 갭투자 시대의 시작
- 2012~2013년부터 갭투자 시작한 사람들의 책이 **2015년에 출간**
- 빨간 책, 노란 책 등 → 출간 시점이 2015년인 이유: 2~3년 전에 투자 시작

---

## 3. 확산기 (2017~2019년)

### 대출 규제 → 풍선효과 발생
- 2017년: 대출 규제 발표 → 그러나 과거 패턴상 풍선효과 예상 가능
- **강남 누르니까 강북으로** (특히 신축 선호)
- 마용성으로 확산 → 구축도 상승
- **10년 넘은 아파트도 입지 좋으면 상승**

### 강사의 첫 번째 상처
- 2017년 10월 첫 강의: "강남 어려우면 분당은 어떠세요?" → 욕먹음
- 당시 분당은 리모델링 논의만 있었고 관심 밖
- 용인 수지에서도 비슷한 경험 → 결국 나중에 모두 상승

### 경기도로 확산 (2019년)
- 회복장(2013)부터 경기도 확산(2019)까지 약 **7년 소요**
- 대부분의 사람들은 이 7년 동안 관심 없다가 뒤늦게 참여

---

## 4. 급등기 (2019~2021년)

### 꿈수(꼼수) 찾기 시작
- 법인 투자, 지방 투자, 대체 상품 등
- **공시지가 1억 미만** → 왜 1억 미만? 좋아서가 아니라 안 좋아서
- 눈이 돌아가서 가는 것 → **탈출 신호**

### 지방 확산 경로 (실제 패턴)

```
서울 강남 → 마용성 → 노도강 → 의정부 → 양주 → 동두천 (끝)
                                          → 포천 아이파크?? (부룡: "동두천이 끝입니다")

남쪽: 수원·용인 → 화성·오산 → 평택 → 여주 (끝)

전북: 전주 → 군산·익산 → 김제 → 정읍 (끝)

강원: 원주·춘천 → 강릉 → 속초(세컨하우스) → 동해·삼척 (끝 = 장렬히 전사)
```

### 위축 신호
- **2021년 2월**: 가계대출 금리 인상 발표
- **2021년 8월**: 대출 총량 규제, 전세대출 중단
- 이 시기에 공시지가 1억 미만 지방 투자 → **가장 위험한 투자**
- 강남은 잘 팔리지만, 공시지가 1억 미만은 팔리지 않음

## 5. 스팟(Spot)과 플로우(Flow) — 핵심 분석 프레임

### 개념 정의

| 구분 | 의미 | 특징 |
|------|------|------|
| **스팟(Spot)** | 특정 시점의 모습 | 뉴스 기사 하나, 특정 월의 가격 변화 |
| **플로우(Flow)** | 흐름의 모습 | 수개월~수년에 걸친 추세 |

### 왜 중요한가
- 대부분의 사람들은 **스팟(뉴스 기사 하나)**을 보고 상승장/하락장을 판단
- 그러나 스팟으로는 미래를 알 수 없음
- **플로우(흐름)를 봐야 시장의 방향을 알 수 있음**

### 세종시 사례 (실전 적용)
- 2025년 5월: "세종시 상승 거래 확대" (스팟: 상승)
- 2025년 7월: "세종시 하락" (스팟: 하락)
- 2025년 9월: "세종 우석타운 신선을 되나" (스팟: 또 다른 기사)
- **플로우로 보면**: 수년간 하락 후 약간의 회복 → 아직 본격 상승 아님
- 투자 판단: **스팟이 안 좋을 때(하락 기사) 들어가는 게 최적**

---

## 6. 현재 시장 위치 분석 (강사 견해)

### "지금은 상승장인가?"

> **강남이 오르는 것 ≠ 상승장**

- 과거 침체장(2010~2012년)에서도 강남은 계속 상승 기사가 나왔음
- 2008년 금융위기 직후, 2011년 유로존 위기에서도 강남은 "좀비처럼" 버팀
- **강남 기준으로 상승장/하락장을 판단하면 안 됨**

### 진짜 침체의 기준
- 외곽·지방의 물건이 거래되지 않는 시장 = 침체장
- 이자 못 버텨 경매로 가는 건수가 11년 만에 최대 (2024년)
- 11년 전 = 2013년 = 침체에서 회복으로 넘어오는 시기

### 강사의 포지션
- 현재는 **침체 또는 정체장**으로 판단
- 장기적으로는 상승 → 전세가격 상승이 계속되고 있으므로
- 그러나 단기적으로 대출 레버리지로 진입하는 것은 위험

### 전세가격 상승 → 다음 상승장의 신호

```
전세가격 상승 (지금 진행 중)
→ 정부 규제 완화 (이재명 정부 후반부 예상)
→ 매매 시장 정상화
→ 본격 상승장 진입
```

- 과거 2010년부터 2015년까지 **5년의 회복장**을 거침
- 현재 2021년 8월부터 약 **4년 경과** → 앞으로 1년 내외 남았을 가능성

## 7. 4교시 핵심 요약

| 핵심 포인트 | 설명 |
|-------------|------|
| **침체기에도 강남은 움직인다** | 좋은 입지는 어떤 상황에서도 먼저 반응 |
| **상승 신호 = 분양시장 경쟁률** | 매매시장보다 분양시장이 먼저 반응 |
| **스팟 vs 플로우** | 뉴스 하나로 판단하지 말고 흐름을 봐라 |
| **7년의 회복장** | 침체에서 확산까지 상당한 시간 소요 |
| **추격매수 가장 위험** | 특히 대출 레버리지 높은 초보자 |
| **전세가격 상승 = 핵심 신호** | 매매 전환 수요의 구조적 전조 |
| **호흡을 길게** | 부동산 투자는 단타가 아닌 긴 호흡의 게임 |
