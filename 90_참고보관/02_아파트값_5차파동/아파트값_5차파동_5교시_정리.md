# 아파트값 5차 파동 특강 - 5교시 정리

## 📌 핵심 주제: 지방장의 두 가지 흐름 + 강사의 실전 경험 + 최종 메시지

---

## 1. 현재 시장 정리

### 단기 vs 장기 시각
- **단기(스팟)**: 강남 상승, 구축 래미안 상승, 마용성 확산 → 뉴스 기사 수준
- **장기(플로우)**: 아직 풀로우 상으로 명확한 방향 확정 어려움
- 강사의 판단: **전세가격 상승이 구조적으로 만들어지고 있는 것**에 주목

### 전세 시장 불안의 구조적 의미
- 전세 시장 불안 지속 → 정부의 규제 완화 가능성
  - 서울·수도권 규제 완화, 또는
  - **지방 규제 완화**로 연결될 수 있음
- 어느 쪽이든 투자자에게 기회 발생

## 2. 지방장의 두 가지 상승 유형

### 유형 1: 서울·수도권 대파동의 풍선효과

> 서울 수도권 상승 → 규제 → 풍선효과로 지방까지 확산

#### 이번 6차 파동의 실제 확산 경로 (2019~2021년)

```
2019.11 부산 조정지역 해제
→ 서울 투자자 "무지 마쇼핑" (부산 해운대)
→ 부산 뒤집어짐
→ 인구 많은 곳 → 차례로 확산
→ 청주 (미분양 급감) → 충주·제천
→ 강원도: 원주·춘천 → 강릉 → 속초 → 동해·삼척
→ 전남: 광주 → 순천
→ 경북: 대구 → 포항 (남구=규제, 북구=비규제 → 북구로 몰림)
→ 경남: 창원 → 김해 → 거제 → 통영
→ 충남: 천안·아산 → 당진·서산 (위험 구간)
```

#### 끝물의 신호
- **대체 상품(다세대, 빌라) 유행** → 끝물
- **공시지가 1억 미만 지역으로 몰림** → 끝물
- 포항 비규제(북구) 같은 입지 나쁜 곳 진입 → 위험
- 이런 패턴이 나타나면 **투자를 멈춰야 하는 시점**

### 유형 2: 지방 활성화 정책에 따른 독자적 상승

> 서울 수도권 침체기에, 정부가 지방을 살리기 위해 규제 완화 → 지방 독자 상승

#### 2008~2014년 사례

| 시기 | 사건 | 효과 |
|------|------|------|
| 2006년 | 지방-수도권 차등 적용 요구 기사 | 지방 아웃성 시작 |
| 2008.1 | 투기지역 해제 + 전매제한 해제 | 분양권 투자 가능 |
| 2008.8 | 양도세 감면 | 투자 유인 |
| 2008년 | 지방 미분양 정부 매입 | 직접 개입 |
| 2009년 | 부산 회복 시작 (전세가 상승) | 공급 부족 |
| 2009.12 | 지방 1순위 청약 조건 완화 (6개월) | 분양 수요 증가 |
| 2010년 | 대구 회복 시작 | 전세가 상승 |
| 2010년 | 미분양 양도세 감면 1년 연장 | 지속적 지원 |
| 2011년 | **지방 전체 폭발** | 본격 상승 |
| 2011~2014 | 지방 미친 듯이 상승 | **규제 없는 상승** |

#### 왜 규제가 없었는가?
- 서울·수도권이 죽어 있으니 → 평균적으로 시장이 "괜찮아 보임"
- 정부가 지방을 규제할 이유가 없었음
- **지방 투자자들이 이 시기에 엄청나게 돈을 벌었음**

### 강사의 판단: 현재는 유형 2의 가능성이 높다

| 현재 지방 상황 | 과거 유사 시점 |
|----------------|----------------|
| 미분양 급증 | 2006~2008년 |
| 악성 미분양 | 2007~2008년 |
| 양극화 심화 | 2006~2008년 |
| 정부 대책에 냉담 | 2007년 |
| 이재명 대통령 "지방 금리" 언급 | 2008년 지방 규제 해제 직전 |

> **부산 해운대 → 수영구 → 동래구 → 남구 순으로 회복 가능**
> **창원 성산구·의창구 → 순차적 확산**

## 3. 전세가격 상승 — 모든 파동의 공통 신호

### 1차~6차 파동 전세가격 상승 정리

| 파동 | 시기 | 전세가율 수준 | 특이사항 |
|------|------|---------------|----------|
| 1차 | 1971년 | 70% | 침체기 첫 전세 상승 |
| 2차 | 1975~77년 | 70%+ | 소형 아파트 중심 |
| 3차 | 1979~81년 | 50→70%+ | 장영자 사건으로 가속 |
| 4차 | 1984~87년 | 78~80%+ | 3저 호황과 결합 |
| 5차 | 2000~01년 | 68.7% → 80%+ | 외환위기 후 전세 폭등 |
| 6차 | 2010~15년 | 50→60→70→90% | 임대차 3법 전 |

### 현재 (2025년)
- 전세가격 상승 **구조적으로 진행 중**
- 서울·수도권 전세 불안 지속
- 부산 등 지방도 공급 부족으로 전세가 상승 시작
- 이재명 정부 후반부에 규제 완화 예상 → **본격 상승 전환 가능**

## 4. 7차 파동 전망

- 6차 파동에서 연속일 수도, 새로운 7차 파동일 수도 있음
- 현재 **전세가격 상승 → 출발점**에 위치
- 서울 외곽 + 경기도 중급지 = **자금 부족한 투자자에게 좋은 기회**
- 조금만 더 기다리면 갭투자 가능 시기 도래 예상

## 5. 강사의 실전 투자 경험담

### 경험 1: 부천 한국아파트 (1997년) — 1교시 답변

| 항목 | 한국아파트 (신축) | 성경아파트 (구축) |
|------|-------------------|-------------------|
| 가격 | 8,900만 원 | 7,600만 원 |
| 특징 | 계단식, 역세권 | 복도식 |
| 결과 | 나중에 9,500 | 나중에 9,300 |

- 신축이 먼저 오르고, 구축이 따라붙는 패턴
- **시장 상승 초입에는 싼 것(구축)이 먼저 오르고, 이후 개별 물건의 본질적 가치로 수렴**
- 못 기다리고 팔았더니 → 그 후 6,000만 원 추가 상승

### 경험 2: 목동 vs 프룬마을 (2001년 결혼 시)

| 항목 | 목동 11단지 | 프룬마을 아파트 |
|------|-------------|-----------------|
| 가격 | 1억 2,000만 원 | 1억 1,500만 원 |
| 차이 | 500만 원 | 신축, 리모델링 |

- 500만 원 차이인데 **신축을 선택** → 이후 목동은 재건축 시대로 가격 폭등
- 남은 돈으로 **아내 차 구입** → 지금 생각하면 통탄
- **신축이 전부가 아님** — 재건축 가능한 구축의 미래가치를 놓친 사례

### 경험 3: 프룬마을에서 발견한 갭투자 기회
- 프룬마을 바로 옆 신축 아파트가 **미분양** 상태
- 전세가격 상승기에 **500만 원이면 갭투자 가능**했음
- 그러나 당시에는 그 방법을 몰랐음
- **모르는 나 자신이 문제였다** — 어떤 곳에서도 기회는 있음

### 경험 4: 고양시 능곡 — 바닥에서의 재기 (2003~2004년)

- 분양권 전매 금지로 모든 것을 잃고 **경기도 능곡으로 이사**
- 서울 주민등록이 경기도로 바뀌는 좌절감
- 그러나 이곳에서 **뉴타운 가능성 발견** → 빌라 매집 (1,000~1,600만 원)
- 이후 수익 실현 → 재기의 발판

### 경험 5: 군산 나운주공 3차 (마지막 투자, 2020년)

| 항목 | 내용 |
|------|------|
| 매수가 | 3,800만 원 (나운주공 3차, 5층) |
| 매도가 | 비공개 (수익은 크지 않음) |
| 최고가 | 이후 1억+ 까지 상승 |

- **시장 흐름을 알고 들어갔지만, 매도 타이밍은 못 맞춤**
- 군산방에서 정체가 들통나 급하게 매도
- 핵심 교훈: **매도 타이밍은 신의 영역, 그러나 마이너스를 친 적은 없다**

## 6. 강사의 투자 철학 종합

### 성공의 비결
> **"최고의 승률보다, 확실하게 돌 통장에 남은 게 최고"**

| 원칙 | 설명 |
|------|------|
| 사이클 이해 | 큰 흐름을 알면 진입·퇴출 판단 가능 |
| 마이너스 무지 | 크게 못 벌어도 잃지 않는 것이 생존 |
| 긴 호흡 | 단타 X, 사이클 투자는 기본 5~7년 |
| 두루두루 경험 | 성공·실패 모두 필요, 구루가 되려면 |
| 레버리지 주의 | 대출 극대화는 하락장에서 생존 불가 |

### 허접했던 투자들이 가르쳐준 것
- 신축만 고집 → 구축 상승 놓침
- 분양권에 올인 → 정책 한 방에 전멸
- 목동 포기 → 재건축 대박 놓침
- 군산 빨리 팔아야 해서 수익 적음
- **그러나 한 번도 시장에서 퇴출되지 않았음** → 이것이 진짜 성공

## 7. 최종 메시지

### 전세가격 상승을 허투루 보지 마라
- 당장 내일이 아니더라도, 1~2년 안에 **엄청난 기회가 될 것**
- 1차 파동부터 6차 파동까지 **한 번도 예외 없이** 전세가격 상승 후 매매가 상승

### 투자 시점의 핵심 신호 3가지

| 순서 | 신호 | 의미 |
|------|------|------|
| 1 | **전세가격 움직이면** | 상승 신호에 가까워짐 |
| 2 | **세금 혜택 부여 (양도세 면제 등)** | 거의 90점 (출발 직전) |
| 3 | **청약제도 개편 + 실수요자 우대** | 50년간 항상 같은 패턴 |

### 지금 할 일
- 너무 급하지 말 것 — **호흡을 길게**
- 과거 회복장은 5~7년 소요 → 현재 약 4년차
- 서울 외곽·경기도 중급지까지가 **자금 부족한 투자자의 기회 구간**
- 상급지에 무리해서 들어가지 말 것 → 나중에 갭투자로 편하게 진입 가능

> **"수익을 내는 사람보다 끝까지 시장에 남아 있는 사람이 이긴다."**
