# 아파트값 5차 파동 특강 - 2교시 정리

## 📌 핵심 주제: 1차~2차 파동 — 반복되는 패턴의 시작

---

## 1. 2교시 학습 포인트

> **"기억해야 할 것은 연도나 사건이 아니라, 사람들이 어떤 반응을 보였는가"**

- 20년 넘게 공부한 강사도 세부 사건은 기억 못함
- 중요한 것은 **선배들의 행동 패턴** → 우리도 똑같이 행동할 것이기 때문

---

## 2. 1차 파동 (1971~1975년경)

### 배경
- 아파트가 많지 않던 시기 (연탄보일러, 화장실이 실내에 없던 시대)
- 강남 개발 초기 — 반포·영동지구는 단독주택단지로 분양
- 실물경기 침체 상황

### 패턴 ①: 경기 침체 → 정부의 경제 살리기
- 실물경기가 침체되면 정부는 **반드시 부양책**을 냄
- 이것은 **50년간 한 번도 변한 적 없는 패턴**

### 패턴 ②: 침체기 → 전세가격 상승
- 집을 안 사니까 전세 수요 증가 → **전세가율 70%까지 상승**
- 1차 파동부터 이미 이 현상이 나타남

### 패턴 ③: 경기 회복 → 구매 수요 증가
- 살림살이 나아지면 "차라리 집 사는 게 낫지 않나?" → 주택 구입자 증가
- 원자재 가격 상승 → 분양가 인상 우려 → "빨리 사야 한다" 심리

### 패턴 ④: 원자재 가격 상승 → 실물자산 선호
- 물가 상승 → 부동산이 인플레이션 헤지 수단
- 가격이 올라야 안 사는 게 정상인데, 오히려 **"더 오르기 전에 사자"** 심리 작동

### 1차 파동의 핵심 사건들
- **동부이촌동, 여의도 분양 대박** → 아파트 시대 도래
- 분양권 전매를 통한 시세차익 시작
- **양도소득세 신설** → 시장 충격 → 가격 하락
- 공급 과잉 → 가격 하락 (키친 파동 원리)

### 1차 파동에서 확인된 패턴 정리

| 패턴 | 내용 |
|------|------|
| 경기 침체 | 정부는 항상 경제 살리기 시행 |
| 침체기 | 전세가격 상승 (전세 수요 증가) |
| 경기 회복 | 구매 수요 증가 |
| 원자재 상승 | 실물자산 선호 현상 → 집값 상승 지원 |
| 처음 겪는 규제 | 극심한 충격 → 가격 하락 |
| 공급 과잉 | 가격 하락 |

## 3. 2차 파동 (1975~1982년경)

### 2차 파동 전반: 침체와 회복

#### 1975년경: 침체기
- 실물경기 침체 → 잠실 개발, 신반포 분양 시작
- 양도소득세 본격 시행 → 아파트 거래 위축
- **공급 초과 → 가격 하락** (키친 파동 반복)
- 소형 아파트 전세가율 70% → 침체기 전세 상승 패턴 재확인

#### 정부의 규제 완화 순서
> **핵심: 다주택자가 아닌 실수요자(갈아타기)부터 먼저 풀어준다**

- 일시적 2주택자 보유 기간 연장
- 이 패턴은 지금도 동일 — "양도세 풀어달라"고 해도 바로 안 풀림

#### 규제 완화 + 전세가격 상승 = 시장 반등
- 전세가 높은 상태에서 규제 완화 → "지금 사야 돼" 심리 폭발
- **분양시장 과열** → 화곡주공, 화랑아파트 등 강세
- 공식처럼 "규제 완화 = 집값 상승"이 아님 → **당시 상황(높은 전세가율)이 전제조건**

### 2차 파동 중반: 과열과 규제

#### 부가가치세 도입 (1977년)
- 분양가상한제 도입 → 분양가 통제
- 신축 분양가가 싸지니까 → 상대적으로 **구축 아파트 가격 상승**
- 분양가상한제 → 신규 분양 인기 → 구축도 동반 상승 → 풍선효과의 원형

#### 중동 특수 (1977~78년경)
- 유동성 대폭 증가 → 부동산으로 뭉칫돈 유입
- 복덕방(공인중개사 이전) 활성화
- 여의도·동부이촌동 인기 지역에서 강북·단독주택·수도권으로 확산

### 2차 파동 후반: 풍선효과와 규제 강화

#### 풍선효과의 전형
- 투기지역 지정 → 양도세 중과 → 복덕방 조사 → 청약 제한 강화
- **정부 규제는 항상 점진적** (전국 거래 금지 같은 한방은 없음)
- 규제할수록 **다른 곳으로 번짐** = 풍선효과

#### 핵심 교훈
> **"정부와 투자자는 티키타카 관계"** — 규제 → 풍선효과 → 추가 규제 → 또 다른 풍선효과

- 신축 먼저 오르고 → 구축으로 번지고 → 주변 지역으로 확산
- 이 패턴은 **1970년대부터 지금까지 동일하게 반복**

## 4. 2교시 핵심 요약

### 반복되는 핵심 패턴

```
① 경기 침체 → 정부 부양책
② 전세가율 상승 (70~80%)
③ 규제 완화 (실수요자부터)
④ 분양시장 과열 → 구축 동반 상승
⑤ 풍선효과 (핵심지 → 주변 → 외곽)
⑥ 정부 규제 강화
⑦ 지속적 규제 → 결국 가격 하락
⑧ 다시 ①로 순환
```

### 투자자가 기억해야 할 것

| 관점 | 내용 |
|------|------|
| **흔들릴 때 진입** | 단기 3~4년 하락은 장기적으로 기회 |
| **규제는 기회의 전조** | 풍선효과로 다른 곳이 오름 |
| **입지가 최우선** | 어떤 규제에도 좋은 입지의 아파트는 중장기적으로 상승 |
| **돈 없는 사람의 전략** | 상승장 초입에 구축·저평가 지역 진입 |
| **추격매수 금지** | "지금 안 사면 안 돼" = 가장 위험한 심리 |
