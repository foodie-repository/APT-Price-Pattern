# 아파트값 5차 파동 특강 - 3교시 정리

## 📌 핵심 주제: 3차~4차 파동 — 패턴의 고착화와 정부 대응의 반복

---

## 1. 3차 파동 (1979~1984년경)

### 배경: 시장 침체기
- 청약 미달, 미분양 증가
- **유원산업 그룹 부도** → 대우 그룹급의 대형 건설사 붕괴
- 건설사들이 극도로 힘든 시기

### 패턴 반복 ①: 침체 → 정부 부양
- 과세 기준 완화
- 주택경기 활성화를 위한 다양한 정책 시행
- **임대차보호법 개정** — 전세 계약 기간 6개월 → **1년으로 확대**
  - 이로 인해 전세가격 급등 (나중에 89년에 2년으로 또 바뀌며 또 급등)
  - 몇 년 전 임대차 3법(4년)으로 바뀌었을 때도 동일한 현상 반복

### 패턴 반복 ②: 전세가격 급등
- 시장 침체 2~3년간 아무것도 안 올라 → "부동산 끝났어" 심리
- 그러나 전세가율이 50% → 60% → 70%로 꾸준히 상승
- 미분양은 계속 늘어나는데 전세가격만 오르는 이중 구조

### 장영자·이철희 어음 사기 사건의 교훈
- 금융시장 위기 발생 → 은행금리 8%까지 상승
- 은행금리 상승 → 집주인들이 **전세보다 월세가 유리** → 월세 전환 증가
- 월세 전환 증가 → **전세 공급 감소** → 전세가격 추가 상승
- 전세가율 70% 돌파 → "차라리 집 사는 게 낫다" → 매매 수요 전환

> **핵심: 금융시장 문제 → 월세 전환 → 전세 공급 부족 → 전세가 상승 → 매매 전환**
> (현재 상황과도 유사한 구조)

### 유동성 증가와 가격 상승
- 은행금리 대비 월세 수익이 더 높으니 → 뭉칫돈이 부동산으로 유입
- **개포지구** 분양 시작 → 집값 재상승
- 아파트 엘리베이터에 "우리 집은 이 정도는 받아야 합니다" 붙이는 현상 등장

### 3차 파동의 규제와 풍선효과
- 정부 규제책 발표 → 그러나 달아오른 분위기는 바로 안 꺾임
- **산지 개발** 등으로 풍선효과 발생
- **영순위 통장** 프리미엄 급등 → 존재하지 않는 청약통장의 매각 차익까지 과세
- **분양가 현실화** (분양가상한제 완화)
  - 신축 분양가 상승 → 기축 아파트도 동반 급등
  - 이 메커니즘은 현재 마용성 상승 → 동대문·노도강 상승과 동일

### 채권입찰제의 등장
- 로또 분양(분양가와 시세 차이가 큰 경우) 방지 목적
- 국민주택채권을 최고가에 매입한 사람에게 당첨권 부여
- 최근 로또 분양 논란에서도 채권입찰제 재도입 논의

### 목동 신시가지 발표 (1983년)
- 대규모 공급 발표 → 심리 위축 → 가격 하락
- **공급 확대 발표는 상승장의 꺾이는 시점**을 만들어냄
- 1984년 수해(홍수)로 목동 일대가 침수 → 이후 목동 아파트는 2순위 취급

## 2. 4차 파동 (1984~1990년경)

### 침체기 → 전세가율 상승 (다시 반복)
- 개포 우성, 잠실 아시아선수촌 등 좋은 아파트는 분양 관심 높음
- 그러나 시장 전체는 침체 → **전세가율 78~80% 이상**
- 상계 신시가지 분양 → 소형 위주, 미분양 다수

### 3저 호황과 유동성
- **3저 호황** (저유가, 저금리, 저달러) → 유동성 대폭 증가
- 유동성 증가 → 부동산 가격 상승 압력
- 주식시장도 동반 상승 → 주식 수익이 부동산으로 유입

### 분양가 자유화의 위력
- 아파트 분양가를 시장에 맡기겠다는 발표
- 분양가 자유화 → 분양가 상승 예상 → "빨리 청약해야" → 수요 폭발
- 분양가 상승 → 기존 주택 가격도 동반 상승
- **분양가상한제 ↔ 분양가 자유화** 변화 시점이 매우 중요한 투자 신호

### 재건축의 시작
- 주택건설촉진법: **20년 이상 된 아파트 재건축** 허용 발표
- 구축 아파트 → 신축으로 변신 가능 → **노후 아파트 급등**
- 분양가 자유화 + 재건축 허용 → **시장 전체적 급등의 방아쇠**

### 임대차보호법 1년 → 2년 (1989년)
- 전세 계약 기간 2년으로 확대 → **전세가격 급등** (3차 파동과 동일 패턴)
- 정부의 임대차 관련 법 개정은 **매번 전세가격 급등을 초래**

### 규제 강화 → 풍선효과 → 대체투자
- 부동산 규제 강화 → 단지 내 상가, 토지 등 대체투자로 풍선효과
- 8.10 대책 약발 소진 → 규제해도 청개구리처럼 행동
- 정부와 투자자의 티키타카 재확인

## 3. 3~4차 파동에서 확인된 핵심 패턴 종합

### 반복 확인된 패턴

| 순서 | 패턴 | 빈도 |
|------|------|------|
| 1 | 침체기에 전세가격 상승 | 매 파동 |
| 2 | 유동성 증가 → 가격 상승 | 매 파동 |
| 3 | 규제 강화해도 바로 안 꺾임 (풍선효과) | 매 파동 |
| 4 | 분양가 관련 정책 변화 → 기축 가격 연동 | 3차~4차 |
| 5 | 임대차법 개정 → 전세가격 급등 | 3차, 4차 |
| 6 | 대통령 선거 전 개발 공약 → 투기 조짐 | 4차 |
| 7 | 지속적 규제 → 결국 가격 하락 | 매 파동 |

### 투자 판단의 핵심

> **"전세가격 상승이 유독 중요한 이유"**
> - 전세가격 상승은 직접적인 집값 상승 신호가 아니라,
> - **매매 전환 수요를 만들어내는 구조적 전조**이기 때문

## 4. 현재 시장에 대한 시사점 (강사 의견)

| 현재 상황 | 과거 유사 시점 | 시사점 |
|-----------|----------------|--------|
| 강남·마용성 상승 | 4차 파동 과열기 | 추격매수 위험 |
| 월세 전환 증가 | 3차 파동 장영자 사건 후 | 전세 공급 부족 → 전세가 추가 상승 가능 |
| 정부 규제 예고 | 매 파동 중반 | 단기 위축 가능, 그러나 풍선효과도 가능 |
| 분양가 관련 정책 논의 | 4차 파동 분양가 자유화 | 기축 가격에 큰 영향 |
| 이빠이 대출로 진입하는 사람들 | 매 파동 말기 | **가장 위험한 투자자** |
