# 07_투자판단

이 폴더는 `공통 기준`이 아니라 `실제 투자 판단과 후속 질문`을 다루는 결과물을 모아둔 곳이다.

구성:

- `01_최종통합보고서`: 현재 시점의 통합 판단 스냅샷
- `02_shortlist`: 우선 매수 검토 지역을 생활권·단지 단계로 내린 실전 shortlist
- `03_QA`: 사용자 추가 질문과 그에 대한 답변을 주제별 폴더로 누적하는 공간

운영 원칙:

- 통합 보고서는 `기준 판단의 스냅샷`으로 유지한다.
- 새로운 질문은 먼저 `03_QA`의 가장 가까운 주제 폴더에 기록한다.
- 반복 확인된 내용만 통합 보고서나 shortlist에 승격 반영한다.
- `질문 1개 = 파일 1개`로 무조건 쪼개지지 않는다.
- 기본은 `주제별 폴더`이고, 질문이 많아지면 각 폴더 안에 `Q-...md` 파일을 누적한다.
