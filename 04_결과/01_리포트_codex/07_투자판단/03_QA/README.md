# Q&A 인덱스

이 폴더는 추가 질문과 답변을 `주제별 폴더`로 관리한다.

운영 규칙:

- 새 질문은 먼저 가장 가까운 주제 폴더에 추가한다.
- 각 주제 폴더의 `README.md`는 운영 안내용이다.
- 실제 질문이 들어오면 해당 폴더 안에 `Q-...md` 파일을 추가하거나, 짧은 질문은 `README.md`에 바로 누적할 수 있다.
- 답변에는 가능하면 아래 5가지를 같이 적는다.
  - 질문
  - 답변
  - 근거
  - 언제 틀릴 수 있는지
  - 관련 문서 링크
- 특정 주제가 너무 길어지면 `90_archive/`로 이전 버전을 보관하고 새 문서로 이어간다.
