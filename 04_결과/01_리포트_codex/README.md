# 리포트 Codex 인덱스

`01_리포트_codex`는 주제별 폴더로 정리한다.

- `00_공통`: 전처리 요약, 통합분석방법론
- `01_매매시장`: 매매 본문, 2차 심화, 전국 커버판, 대응 방침/요약표
- `02_임차시장`: 임차 본문, 2차 심화, 전국 커버판, 임차 유형/검증표
- `03_정책영향`: 정책 본문, 2차 보강, 전국 커버판, 정책 판정/대조군/토허 표
- `04_공급`: 공급 관련 분석 보고서
- `05_투자검토`: 투자 검토 대상군, 후보 목록 게이트
- `06_예측검증`: 예측 점수, 과거 데이터 검증, 유사국면, 예측·검증 보고서
- `07_투자판단`: 최종 통합 보고서, shortlist, 질문응답형 투자 멘토 노트
- `90_archive`: 구버전 보고서 보관
